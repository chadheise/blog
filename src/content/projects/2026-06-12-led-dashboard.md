![LED Dashboard](/assets/img/2026-06-12-led-dashboard/ledDashboard11.jpg)

- View the project on my [GitHub](https://github.com/chadheise/led-dashboard)
- LED dashboard for displaying sports scores, stocks, weather, world clock, flights overhead, text and more
- Control app for customizing modules and creating playlists
- Expandable framework for adding custom apps and modules
- 3D printed frame
- Runs on Raspberry Pi 4 with [Adafruit RGB Matrix HAT](https://www.adafruit.com/product/2345)
- Custom power circuitry
